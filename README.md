# VSNake

​VSNake is a game where you can compete against a friend using the same keyboard; beyond the basic gameplay, you can customize the match to your liking and encounter new types of food items, each with its own unique effect.

## Foods

- Apple: Increases your size by 1

- Carrot: Increases your speed by 50% for 2.5 seconds

- Pineapple: Increases your size by 2 but reduces your speed by 25% for 2 seconds

- Broccoli: Reduces your size by 2 but increases your speed by 65% ​​for 3 seconds

## Mechanics

​You can change things like the speed at which the snakes move, the maximum amount of food that appears at once, and the timing of their appearance!​

## How I made it

I created this project solo using the Godot game engine; it took about two days to complete, and I hand-crafted all the sprites and mechanics myself.

Before starting, I asked myself what I could build that wasn't overly complex but would still allow me to learn something new—something beyond what I had previously practiced in game development.

The first thing that came to mind was *Snake*!

Then, I thought about what I could add without losing the game's original essence. I decided to introduce new types of food (without complicating the controls) and add a second player for head-to-head combat, ensuring a level playing field for both.

After settling on the concept, I started by creating some test menus (such as the main menu and the game screen itself).

Once the menus were ready and I’d verified everything was working, I began implementing the mechanics—starting with the test map and then moving on to the snake functionality!

After getting the snake mechanics working (along with some basic sprites), I set up the food spawning system and logic; this was relatively easy since I kept the implementation simple.

Towards the end, I polished the user interfaces and improved the sprites for the snakes, food, map, and other elements.

Finally, I designed the lettering for the game icon and the text displayed at the start of the game.
